В трёх директориях представлены конфигурации для извлечения униграмм, биграмм и триграмм из текста. Для запуска необходимо поместить в соответствующую директорию откомпилированную программу парсера Tomita, соответствующую операционной системе. Входной файл input.txt, на выходе файл PrettyOutput.html, в конце которого две таблицы с выделенными фактами -- первая таблица содержит факты в том виде, в каком удобнее извлекать текущей версией парсера. Для примера во всех директориях есть входной и выходной файлы.

В директории Unigram содержится конфигурация для извлечения из текста униграмм следующего вида в нормализованном виде: существительное, прилагательное (не являющееся местоимением либо числительным -- в Tomita все эти части речи идут как прилагательное), наречие, глагол.
Минусы: извлекаются некоторые предлоги, которые обрабатываются как существительные; есть слова в кавычках, слова с большой буквы (не имена собственные).

В директории Bigram содержится конфигурация для извлечения согласованных цепочек из двух слов следующего вида: прилагательное и существительное, существительное и прилагательное, наречие и глагол, глагол и наречие, два существительных, существительное и глагол.
Минусы: из-за извлечения некоторых предлогов в качестве существительных цепочки тоже получаются ошибочными, есть слова с большой буквы (не имена собственные).

В директории Trigram содержится конфигурация для извлечения цепочек вида два прилагательных и существительное.
Минусы: в отличие от биграмм слова в цепочке не всегда согласованы, хотя в качестве главного слова и там, и там, указано существительное.
